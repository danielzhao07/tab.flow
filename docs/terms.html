2. Description of Service

tab.flow is a browser extension that provides a visual tab management overlay, including features such as:

  • A searchable grid view of all open tabs
  • Tab grouping, snoozing, and recently-closed tab recovery
  • Workspace saving, bookmarking, and note-taking
  • AI-powered tab assistant
  • Optional cloud sync across devices (requires account)

The core extension functionality works entirely offline and locally within your browser. Cloud features require creating an optional account.
