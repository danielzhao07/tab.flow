Demo: Full-Screen Grid HUD
Overview

All your tabs in one beautiful grid

Press Alt+Q to open a full-screen overlay showing every open tab as a card. Each card displays the favicon, title, URL, group color stripe, and status badges for pinned, playing, or duplicate tabs.

Demo: Instant Fuzzy Search
Search

Find any tab in milliseconds

Start typing to instantly filter tabs with fuzzy matching powered by Fuse.js. Use smart filters like is:pinned, domain:github.com, or group:work to narrow results even further.

Demo: AI Tab Assistant
AI Powered

Let AI organize your chaos

Type @ to activate the AI assistant powered by Groq. It can group tabs by topic, close old tabs, create workspaces, and handle 21 different actions — all from a single natural language prompt.

Demo: Smart Tab Groups
Organization

Smart grouping, zero effort

Tab.Flow detects when you have multiple tabs from the same domain and suggests groups automatically. One click to organize with smart color assignment for 25+ popular domains like GitHub, Google, and YouTube.

Demo: Workspaces
Workflow

Save your context, switch projects

Save your current set of tabs as a named workspace. Switch between projects without losing your place. Workspaces preserve tab groups and can sync to the cloud across devices.

Demo: Keyboard Navigation
Productivity

Navigate without your mouse

Full 2D arrow key navigation across the responsive grid. Enter to switch, Ctrl+X to close, Tab to multi-select. Every action has a keyboard shortcut — press ? to see the full cheat sheet.

And so much more

Small details that make a big difference.

Tab Suspender

Automatically suspend inactive tabs to free up memory. Pinned and audible tabs are never touched.

Command Palette

Type > in the search bar to access quick commands: close duplicates, group all tabs, export data, and more.

Tab Notes

Attach text notes to any tab. Notes are included in search results so you can find tabs by your own annotations.

Drag to Reorder

Drag tabs in the grid to reorder them. Your custom order persists across sessions.

Multi-Window

See and manage tabs across all browser windows. Move tabs between windows or merge them with one click.

Undo Close

Accidentally closed a tab? An undo toast appears instantly, letting you restore it with one click or Ctrl+Z.
