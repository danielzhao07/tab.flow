Privacy Policy

Effective March 2026 v1.0.0

What We Access

TabFlow accesses the minimum data needed to show and manage your tabs.

  • Tab titles, URLs, and favicons — displayed in the HUD overlay. Never sent to any server.
  • Tab group names and colors — read locally for group labels. Never stored remotely.
  • Recently closed tab metadata — read from Chrome's session API for the "Recently closed" section.
  • Your email — collected only if you create an account via AWS Cognito.
  • Bookmarks, notes, and workspaces — synced to the cloud only when signed in.
  • Settings — stored locally in chrome.storage.local. Optionally synced when signed in.

What Stays on Your Device

This data never leaves your browser:

  • Open tab list, activity order (MRU), and group info — extension memory only.
  • Tab thumbnails — captured and held in memory for the current session.
  • Frecency data — stored in chrome.storage.local on your device.
  • Snooze schedules — stored via chrome.alarms and local storage.
  • HUD search queries — never recorded.

Optional Cloud Sync

Cloud features require sign-in. If you never create an account, nothing leaves your device.

When signed in, the following may be synced:

  • Saved workspaces (named tab sets)
  • Bookmarks saved through TabFlow
  • Notes attached to tabs
  • Extension settings (theme, layout, etc.)

You can delete your account and all cloud data at any time by contacting us.
